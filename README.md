# game_L904_reaction_timer

リアクションタイマーゲーム - 反射神経を測定するシンプルなクリックゲーム

## 🎮 ゲーム概要

「スタート」ボタンを押すと、ランダムな時間（0.8〜3秒）後に画面が緑色に変わります。その瞬間を素早くクリックして、あなたの反応速度を測定しましょう。早押し（フライング）すると失敗になります。

## 🚀 デモ

[GitHub Pagesでプレイ](https://kmh-no3.github.io/game_L904_reaction_timer/)

## 📋 機能

- **リアクションタイム測定**: 緑色に変わった瞬間の反応速度をミリ秒単位で表示
- **フライング検出**: 早押しを検出して失敗として記録
- **シンプルなUI**: 直感的で分かりやすいインターフェース

## 🛠️ 技術スタック

- **TypeScript**: 型安全なJavaScript
- **Vite**: 高速なビルドツール
- **Vanilla CSS**: カスタムスタイリング

## 📦 セットアップ

### 必要な環境

- Node.js 18以上
- npm または yarn

### インストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで `http://localhost:5173` を開いてください。

### ビルド

```bash
npm run build
```

ビルド成果物は `dist/` フォルダに生成されます。

## 🎯 使い方

1. 「スタート」ボタンをクリック
2. 画面が黄色（待機中）になります
3. 画面が緑色に変わったら、すぐにクリック！
4. 反応速度がミリ秒単位で表示されます
5. フライング（緑になる前にクリック）すると失敗として記録されます
